## Architecture

Orchestra Platform consists of four main components working together to provide a seamless workshop experience:

<CardGrid>
  <Card title="Orchestra Operator" icon="star">
    Kubernetes operator managing workshop lifecycle and resource provisioning
  </Card>
  <Card title="Orchestra API" icon="laptop">
    REST API backend handling workshop operations and user management
  </Card>
  <Card title="Orchestra Frontend" icon="document">
    Web application providing an intuitive interface for workshop creation
  </Card>
  <Card title="Orchestra Docs" icon="information">
    Comprehensive documentation and user guides (this site)
  </Card>
</CardGrid>

For more details, see [the architecture page](/architecture/platform-overview/)
